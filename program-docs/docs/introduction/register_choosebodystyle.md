登录成功后，点击“下一步”进入主体信息提交环节，主体认证通过后就可以创建智能小程序。

![图片](../../img/introduction/register/1.1.png)




目前支持的主体类型：媒体、企业、政府、其他组织。主体类型、主体名称一旦提交后将无法更改。



> 暂不支持个人主体类型开发者入驻。
> 当前支持一个主体对应多个账号，但需要每个账号分别进行真实性认证。

![图片](../../img/introduction/register/1.2.png)
